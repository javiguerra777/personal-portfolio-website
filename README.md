## Getting Started

### Prerequisites
- Node.js
- MongoDB
* npm
  ```
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages for both client and server
    ```sh
    cd client
    npm install
    cd ../server
    npm install
    ```
